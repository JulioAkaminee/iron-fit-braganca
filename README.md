# Iron Fit Bragança

**Landing Page de Alta Performance para Crossfit & Academia**

---

## Sobre o Projeto

O **Iron Fit Bragança** é uma landing page imersiva e de alta conversão desenvolvida para uma box de Crossfit e academia localizada em Bragança Paulista. O site foi projetado para transmitir energia, força e modernidade, utilizando um visual **cyberpunk/sport** em **dark mode** que se diferencia das academias tradicionais.

O projeto faz parte do portfólio de demonstração da **Akamine Web Studio**, estúdio de desenvolvimento web sediado em Bragança Paulista.

---

## Destaques

- **Design Cyberpunk/Sport:** Interface dark com neon laranja (#F97316) e verde (#22C55E), tipografia futurista e efeitos visuais marcantes.
- **Animações Ricoas:** Integração com AOS (Animate On Scroll), efeitos glitch, floating elements, scanlines e glows dinâmicos.
- **Imagens Reais do Nicho:** Utilização de fotografias profissionais de atletas, box e equipamentos de Crossfit.
- **Totalmente Responsivo:** Layout adaptado para mobile, tablet e desktop com menu mobile animado.
- **Alta Conversão:** Formulário de captura para aula experimental gratuita, tabela de horários completa e cards de planos destacando o mais popular.

## Stack Tecnológica

| Tecnologia | Uso |
|------------|-----|
| HTML5 | Estrutura semântica e acessível |
| Tailwind CSS (CDN) | Estilização utilitária e responsiva |
| AOS | Animações de entrada ao scroll |
| Lucide Icons | Ícones modernos e consistentes |
| Google Fonts | Orbitron (títulos) + JetBrains Mono (corpo) |
| JavaScript Vanilla | Interatividade, menu mobile e scroll suave |

---

## Seções do Site

1. **Hero** — Full-screen com atleta, estatísticas animadas e CTA duplo
2. **Modalidades** — 6 cards com imagens reais: Crossfit, Funcional, Musculação, HIIT, Mobility, Competição
3. **Planos/Preços** — 3 tiers (Starter, Athlete, Elite) com destaque visual no plano mais popular
4. **Horários** — Tabela semanal completa com filtros por modalidade e turno
5. **Aula Experimental** — Formulário de captação com background imersivo
6. **Footer** — Informações de contato, redes sociais, endereço e créditos

```bash
npx serve .
```
